For more details, including installation and usage instruction, please visit my page:

http://hieudt.info/wp-plugins/quick-slugs/

== Installation ==

   1. Download the plugin and extract its content
   2. Upload the complete folder “quick-slugs” to the “/wp-content/plugins/” directory
   3. Activate the plugin through the ‘Plugins’ menu in WP-Admin
   4. Go to ‘WP-Admin -> Posts -> Quick Slugs’ to edit slugs
